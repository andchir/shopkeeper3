
Changelog:
==========

2.3.1pl2

- При возвате на первую страницу параметр "page" убирается из URL (спасибо пользователю slaad).

- Добавлен метод tagManagerBase::invokeEventCombine(), позволяющий комбинировать выходы от нескольких плагинов.
    Доработана работа с мульти-валитностью.

2.3.1

- Поддержка jQuery 3.x
- Обновление функционала мультивалютности. Курсы валют берутся из настроек Shopkeeper3.
- Другие мелкие исправления и улучшения.

2.3

- Сделана поддержка ручного ввода значений слайдеров.
- В filters.js добавлен параметр "filter_delay" - Задержка до отправления запроса на сервер (сбрасывается после каждой отметки фильтра)
- Исправлена проблема с блокировкой фильтров, которая происходила с задержкой (только filters.js).
- Другие мелкие исправления и улучшения.


2.2rc1

- Сделана поддержка getProducts 1.4+.
- Обновлен AngularJS до версии 1.4.6, Bootstrap до версии 3.3.5, Angular UI Bootstrap до версии 0.13.4.

-------------------------------

2.1

- Решена проблема в случае если у контейнера отключен суффикс "/".
- В админке решена проблема со стилями, которая возникала из-за bootstrap.css.
- При переходе по истории отмечаются соответствующие чекбоксы фильтров.
- Сделан авто-скролл наверх при применении фильтров.
- Исправлена ошибка в cron_updatefilters.php, из-за которой не корректно работало обновление значений фильтров (https://bitbucket.org/andchir/tagmanager2/issue/4/assets-components-tag_manager2).
- В админке исправлена проблема с позиционированием всплывающего окошка при закрепленном верхнем тулбаре.
- Сделана авто сортировка значений фильтров по алфавиту.
- Исправлено https://bitbucket.org/andchir/tagmanager2/issue/1/tm2
- Обновлен AngularJS до версии 1.3.13 и Angular UI Bootstrap до версии 0.12.0.
